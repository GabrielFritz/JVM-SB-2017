void init_methodexecution ( execution *  e,
char *  class,
char *  method,
char *  descriptor,
int  args 
)

Prepara o frame para a execucao de um metodo.

Parâmetros
[in]eInterpretador
[in]classNome da classe
[in]methodNome do Metodo
[in]descriptorDescritor do Metodo
[in]argsArgumentos de Entrada do metodo

Primeiro, e' checada se a classe ja' esta' carregada. Em seguida, um novo frame e' inserido na pilha de frames e iniciado com os dados do metodo a ser executado. Para cada argumento, e' criada uma entrada na pilha de operandos e na pilha de tipos. Por fim, os operadores sao colocados no array local da pilha de frames do interpretador.
